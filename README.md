# Hollowbrook Press Website

Free static website for GitHub Pages.

## To publish
1. Create a GitHub repository named `hollowbrookpress` or `hollowbrookpress.github.io`.
2. Upload all files in this folder.
3. Go to Settings > Pages and publish from the main branch.
4. In your domain registrar DNS, point `hollowbrookpress.com` to GitHub Pages using GitHub's current custom domain instructions.
5. Replace the placeholder Amazon `#` links in the HTML files with your real book URLs.

The CNAME file is already set to `hollowbrookpress.com`.
